## Installation
Add it in your root build.gradle at the end of repositories:

```groovy
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
build.gradle (app)
```groovy
dependencies {
    implementation 'com.github.md0092651:PlotWizard:0.0.2'
}
```
